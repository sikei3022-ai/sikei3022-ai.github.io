# Skride — сценарное тестирование

**Стенд = живое приложение:** https://sikei3022-ai.github.io/
**Сервер:** https://skride-server.onrender.com (первый запрос «будит» сервер ~30–50 сек)

Совет: для быстрого наполнения данными нажми **Настройки → 🧪 Тестовые данные**. Очистить — там же «🗑 Очистить всё».

---

## 1. Первый запуск / карусель
- [ ] Открыть приложение в режиме инкогнито (чистый старт).
- [ ] Появляется карусель из 6 слайдов: бег+вело, ИИ-тренер, легенды, 2 фото про пульс, статистика.
- [ ] Кнопки «Пропустить» и «Далее/Начать» работают.
- [ ] Сразу выдана случайная смешная аватарка.

## 2. Пробный период (14 дней)
- [ ] Открыть ⭐ — статус «Пробный период: осталось 14 дней».
- [ ] Кнопка ⭐ сверху показывает «14 дн. бесплатно».
- [ ] Цена в окне Premium — «100 ₽/мес».
- [ ] Все функции работают в триале без активации Premium.

## 3. Профиль
- [ ] Изменить имя, город, пол — сохраняется (карта НЕ вылезает поверх окна).
- [ ] «🎲 Смешная» — меняет аватарку. Загрузка фото — работает.
- [ ] Пол меняет фото тренера (бег) и велотренера (вело).

## 4. Запись бега (GPS)
- [ ] Активность «🏃 Бег». Режим GPS. «Старт» → идёт время/дистанция → «Стоп» → сохранить.
- [ ] Тренировка появилась в «Активностях» и в «Эта неделя».

## 5. Ручной ввод + дата
- [ ] Настройки → режим «Ручной ввод».
- [ ] На карте поставить точки маршрута; линия «Волнистая/Прямая» переключается.
- [ ] Поле «Дата» открывает календарь (число/месяц/год); рядом показывается день недели.
- [ ] Дистанция и название сохраняются с выбранной датой.

## 6. Велосипед
- [ ] «🚴 Велосипед» → появляются типы велосипеда; раздел уровня → «Уровень велосипедиста».
- [ ] Баннер тренера — велоспорт (фото по полу). Вопросы-подсказки — велосипедные.
- [ ] «Легенды велоспорта» — открывается список, виден твой рейтинг по скорости.

## 7. Карта
- [ ] «Карта приложения» (OSM) и «Яндекс карта» переключаются.
- [ ] Кнопка 📍 на карте определяет геопозицию.
- [ ] Нет украинского флага в подписи.
- [ ] «Снимок маршрута» сохраняет картинку с картой-подложкой.

## 8. ИИ-тренер
- [ ] Настройки → адрес сервера вписан, «Проверить» → «Сервер на связи».
- [ ] Вопрос текстом → развёрнутый ответ (облако). Голос проигрывается.
- [ ] Кнопки-подсказки и режим «Разговор» работают.

## 9. Устройства
- [ ] Карточки: дорожка, умные часы, умный браслет — кнопки подключения по Bluetooth.
- [ ] (на телефоне с Chrome) запрос Bluetooth-устройства появляется.

## 10. Статистика
- [ ] 📊 (сверху или в профиле) открывает отдельную страницу, «‹» возвращает.
- [ ] Видны: общие плитки, бег, велосипед, ходьба, суммарно (спорт/телефон/всего).
- [ ] Включить ходьбу, вписать км → пересчитывается «Всего».

## 11. Аккаунт и облако
- [ ] Настройки → 💾 Хранение → «☁️ В облаке».
- [ ] Регистрация (почта+пароль) → «Вы вошли», «синхронизировано ✓».
- [ ] Добавить тренировку → уходит в облако.
- [ ] Статистика → реальный рейтинг «место X из Y».
- [ ] Выйти → данные остаются на телефоне. Войти заново → данные подтянулись.
- [ ] Проверить на втором устройстве/браузере: вход тем же аккаунтом → те же тренировки.

## 12. Языки
- [ ] Настройки → Язык: Русский / English / Italiano / 日本語 — заголовки разделов меняются.

## 13. Конец триала (проверка логики)
- [ ] (для теста) можно временно поставить дату телефона на +15 дней → статус «Пробный период завершён», платные функции просят Premium.
- [ ] «Оформить подписку — 100 ₽/мес» (демо) включает Premium.

---

### Что считать багом
- Карта поверх модальных окон.
- Тренер молчит при подключённом сервере и Premium/триале.
- Данные пропадают при выходе/входе в облачный аккаунт.
- Снимок без карты-подложки.
